How to run it:

LOCALLY:
- Clone this repository into to your local computer.
- $ gem install bundle
- $ bundle
- $ cd Tic_Tac_Toe_8th_Light
- $ cd lib
- $ ruby setup.rb
- Use in irb

    Specification:

    I was asked to implement the following features in a classic tic-tac-toe game:

    - Three modes of game: Human vs Human, Human vs Computer, Computer vs Computer;
    - Human player can pick their name and sign(marker);
    - In all modes users are asked to choose who goes first;
    - Computer plays on a medium level - each time it is to make a move, it:
    - randomizes whether to make a random move or a best possible move;
    - makes a move selected in the step above;
    - Command-line interface;

    Additions:
    - Goblet for the winner
    - Colored board and error messages to make the flow of the game easier to follow;
