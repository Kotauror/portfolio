Bookmark Manager
Pair programming
A ruby application created as a weekly challange at week 4 of Makers Academy.
Collect, comment and tag bookmarks.
Front-end: CSS, HTML,
Back-end: Ruby, SQL,
Testing: Rspec,
Database: Postgresql,
Framework: Sinatra

How to run it:

LOCALLY:
- Clone this repository into to your local computer.
- $ gem install bundle
- $ bundle
- $ rackup
- visit http://localhost:9292/

Design of code:

Class Link

methods:

  • @id - id of a comment.
  • @url - url of a bookmark.
  • @title - title of bookmark.
  • self.all - gets all bookmarks from the one-to-many database.
  • self.add - creates a new bookmark and saves it in the database.
  • self.delete - deletes bookmark from the database.
  • self.update - updates a bookmark in the database.
  • self.is_url?(url) - cheks if provided link starts with www or http://.
  • self.is_title?(title) - cheks if database contains a title

    • Class Comment

    methods:

  • @id - id of a comment.
  • @text - text of a comment.
  • @link_id - id of a bookmark to which the comment refers.
  • self.all - gets all comments from the database.
  • self.add - creates a new comment and saves it in the database.
  • self.is_id?(id) - checks if an id of a bookmark exists.

    • Class Tag

    methods:

  • @id - id of a tag.
  • @content - content of a tag.
  • self.all_pairs - gets all bookmark-tag pairs from the many-to-many database.
  • self.all_tags - gets all tags from the database.
  • self.add_to_tags(tag) - adds a tag to tags database.
  • self.add_to_link_tags(link_id, tag_id) - adds a pair to many-to-many database

    • Class DatabaseConnection

    methods:

  • self.setup(database) - creates a connection using PG gem
  • self.connection - returns the connection
  • self.query(sql_string) - a skeleton for writing SQL commands

    • User Stories:

    As a time-pressed user
    So that I can quickly go to web sites I regularly visit
    I would like to see a list of links on the homepage

    As a time-pressed user
    So that I can quickly find web sites I recently bookmarked
    I would like to see links in descending chronological order

    As a time-pressed user
    So that I can save a website
    I would like to add the site's address and title to my bookmark manager

    As a time-pressed user
    So that I can organise my many links into different categories for ease of search
    I would like to add tags to the links in my bookmark manager
