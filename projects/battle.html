How to run it:

LOCALLY:
- Clone this repository into to your local computer.
- Run $ gem install bundle
- Run bundle
- Run $ rackup

ONLINE:
- http://epic-battle.herokuapp.com/

Design of code:

Class Game

methods:

  • self.create - creates a new game
  • self.instance - returns an instance of game created in self.create
  • @players - players of a current game
  • @current_turn - tells which player has a turn
  • attack(player) - attacked player receives damage
  • heal(player) - healing player receives additionl points of life
  • switch_turns - changes the value of @current_turn
  • defending_player - returns the player which is currently in the defence
  • game_over? - tells if the game is still on

    • Class Player

    methods:

  • @name - name
  • @hp - hitpoints (100 by default)
  • receive_dagame - decrease the hp by random number 1-20
  • heal - increase the hp by random number 1-10

    User Stories:

    As two Players,
    So we can play a personalised game of Battle,
    We want to Start a fight by entering our Names and seeing them

    As Player 1,
    So I can see how close I am to winning
    I want to see Player 2's Hit Points

    As Player 1,
    So I can win a game of Battle,
    I want to attack Player 2, and I want to get a confirmation

    As Player 1,
    So I can start to win a game of Battle,
    I want my attack to reduce Player 2's HP by a random number

    As two Players,
    So we can continue our game of Battle,
    We want to switch turns

    As Player 1,
    So I can see how close I am to losing,
    I want to see my own hit points

    As Player 1,
    So I can lose a game of Battle,
    I want Player 2 to attack me, and I want to get a confirmation

    As Player 1,
    So I can start to lose a game of Battle,
    I want Player 2's attack to reduce my HP by a random number

    As a Player,
    So I can Lose a game of Battle,
    I want to see a 'Lose' message if I reach 0HP first

    As a Player,
    So I can extend my joyous experience of Battle,
    I want to have an attack that heals some of my Hit Points

    As a Player,
    So I can enjoy my game of Battle,
    I want a sexy user interface
