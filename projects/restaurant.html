Takeaway restaurant simulator
Solo project
A ruby program created as a weekend challange at the 2nd weekend of Makers Academy.
Order food and receive a SMS confirmation.
Back-end: Ruby,
Testing: Rspec

How to run it:

- Clone this repository into to your local computer.
- Run $ gem install bundle
- Run bundle
- Use in irb

Design of code:

Class Restaurant

methods:

  • @menu - array of available dishes.
  • create_dish(name, price) - create a dish.
  • display_menu - shows a menu in a structured way.
  • create_order(dish_list, total_number) - creates an order according to the names of dishes and a total number of dishes diven by the client. The total number is used to confirm that the user will receive a correct number of dishes.

    • Class Dish

    method:

  • @details - hash having a name and a price of a dish.

    • Class Order

    methods:

  • @list_of_items - list of dishes in an order.
  • @total_number - total number of ordered dishes given by the user.
  • add_items(name, price, amount) - adds dishes to the list_of_items.
  • order_info - informs of the final price and number of dishes.
  • count_final_price - counts final price.
  • count_dishes - counts number of dishes in user's order.
  • send_order - sends order if the number of dishes is correct. Creates a new Message.

    • Class Message

    methods:

  • creates a new Message with ETA.

    • User Stories:

    As a customer
    So that I can check if I want to order something
    I would like to see a list of dishes with prices

    As a customer
    So that I can order the meal I want
    I would like to be able to select some number of several available dishes

    As a customer
    So that I can verify that my order is correct
    I would like to check that the total I have been given matches the sum of the various dishes in my order

    As a customer
    So that I am reassured that my order will be delivered on time
    I would like to receive a text such as "Thank you! Your order was placed and will be delivered before 18:52" after I have ordered
