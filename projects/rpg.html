How to run it:

LOCALLY:
- Clone this repository into to your local computer.
- $ ruby meow_meow_land.rb

Design of code:

Class Character

methods:

  • @name - name
  • @strength - integer
  • @intelligence - integer
  • @luck - integer
  • @health - integer
  • change_strength - changes the strength
  • change_health - changes the health

    • Class Team

  • @characters - array of picked characters
  • show_teammates - shows the team
  • has_character(character_name) - checks if @characters has a particular character
    •
  • remove_character(character)- removes a character from a team
  • remove_character_at(index)- removes a character at a particular index from a team
  • get_character(letter) - picks the character according to a letter given by a player
  • team_size - size of a team

    • Class Monster

    methods:

  • @name - name
  • @strength - integer
  • @intelligence - integer
  • @luck - integer
  • @health - integer
  • change_strength - changes the strength
  • change_health - changes the health
