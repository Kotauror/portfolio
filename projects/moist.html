Notepad
Group project
A javascript application and testing framework created as a weekly challange at week 7 of Makers Academy.
Write notes and display them on website without the use of database.
Front-end: Javascript, HTML, CSS
Testing: Moist

How to run it:

The notepad:
LOCALLY:
- Clone this repository into to your local computer.
- Use open the index.html file in the browser
ONLINE:
- moist.surge.sh

The testing framework:
- npm install moisty (learn more about moisty here )

Design of js code:

Constructor Notepad

methods:

  • content - array of notes

    • Constructor Note

    methods:

  • content - content of a note
  • id - id of a note

    • Other methods:

  • sendData(note) - creates a Note object and adds it to the Notepad
  • displayNote(noteObject) - displays the content of a particular note on the website
  • _longNote, _shortNote, _appendChild, _onClick - methods that faciliate the display of notes

    • User Stories:

    As a programmer
    I can see a list of my notes, where each note is abbreviated to the first 20 characters
    So I can find the one I want

    As a programmer
    I can create a new note
    So I can record something I need to remember

    As a programmer
    I can see the full text of an individual note on its own page
    So I can see all the information in the note
