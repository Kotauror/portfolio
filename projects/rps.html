Rock-Paper-Scissors [Solo Project]
A ruby application created as a weekend challange at the 3nd weekend of Makers Academy.
Play Rock Paper Scissors!
Front-end: CSS, HTML,
Back-end: Ruby,
Testing: Rspec,
Framework: Sinatra

How to run it:

LOCALLY: - Clone this repository into to your local computer.
- Run $ gem install bundle
- Run $ bundle
- Run $ rackup

ONLINE
- rockspapersscissors.herokuapp.com

Design of code:

Class SingleGame

methods:

  • self.create - creates a new class instance.
  • self.instance - returns the class instance of the game class.
  • computer_weapon - picks the weapon for the computer.
  • calculate_result - returns result based on rules:
    Rock beats scissors
    Scissors beats paper
    Paper beats rock.
