Makers BNB [Group Project]
A javascript application created as a weekly challange at week 6 of Makers Academy.
Add a place, sing up, log in, log out.
My first contact with asynchronous functions.
Front-end: Javascript, CSS, HTML
Back-end: Node.js, Sequelize,
Testing: Capybara, Zombie,
Database: Postgresql

How to run it:

- Clone this repository into to your local computer.
- $ brew install node
- $ npm install
- $ node.js
