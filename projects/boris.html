Boris Bikes
A command line ruby application created as a weekly challange at week 1 of Makers Academy.
programming language: Ruby, testing: Rspec

User Stories:

As a person,

So that I can use a bike,
I'd like a docking station to release a bike.

As a person,
So that I can use a good bike,
I'd like to see if a bike is working

Classes:

Bike

methods:

  • 1) working? - tells if a bike is working.
    • 2) report - returns a string telling if the bike is working.

    Docking Station

    methods:

    1) @bikes - an array of bikes available at a station 2) @capacity - capacity of a station 3) release_bike - removes first possible working bike from bikes array 4) dock - adds a bike to a docking station if there are free places 5) count_of_broken_bikes - returns number of broken bikes at a station 6) empty_station_of_broken_bikes - removes broken bikes from an array 7) full? - checks if station is full 8) empty? - checks if station is empty

    Garage

    methods:

    1) collection_of_bikes - array of bikes in the garage 2) receive_bikes - picks bikes from van: adds them to collection and removes them from a van 3) fix_bikes - changes the status of a bike to working.

    Van

    methods:

    1) collection_of_bikes - array of bikes in the van 2) empty? - tells if van is empty 3) pass_bikes - returns the collection of bikes 4) delete_bikes - empties the array of bikes
