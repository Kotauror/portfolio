How to run it:

ONLINE:
- photo-fridge.herokuapp.com

LOCALLY:
- Clone this repository into to your local computer.
- $ gem install bundle
- $ bundle
- $ rake db:create
- $ rake db:migrate / If you are having problems with running rake db:migrate try running bin/rake db:migrate;
- $ rails server
- Visit localhost:3000;
